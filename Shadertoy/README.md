# Shadertoy GLSL presets :

### Copy those files into the Shadertoy presets folder. Add the lines from the Shadertoy.txt file into your own Shadertoy.txt.

## Available presets :
- <img src='Shadertoy/icons/AFX_DeSpill.frag.png' width='32'> **[AFX_DeSpill]** : Based on the Despill algo if green is greater than the average of the red and blue channels, then bring green down to that color... also works with blue and redscreen, this algo is excellent for maintaining skintones...
