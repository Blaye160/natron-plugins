# FXAA_GL

1) Add the following line to the Shadertoy.txt file :
   "Effect/FXAA" "FXAA-natron.frag.glsl"

2) Copy the FXAA-natron.frag.glsl file to the Shadertoy folder.

3) Copy the FXAA_GL.py and FXAA.png files to your PyPlug folder.

Done
